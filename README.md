# Learning NumPy

A very simple and easy guide to get started with NumPy. This repository contains my personal learning notes, examples, and small explanations to understand NumPy fundamentals.

## Contents

1. **1-list-vs-nparray.ipynb** – Difference between Python lists and NumPy arrays
2. **2-creating-list.ipynb** – Creating arrays in NumPy
3. **3-array-properties.ipynb** – Understanding properties of arrays
4. **4-operations-on-arrays.ipynb** – Basic operations on NumPy arrays
5. **5-copy-vs-view.ipynb** – Copy vs View behavior in NumPy
6. **6-datatypes-in-numpy.ipynb** – NumPy data types explained
7. **7-axes-in-multidimentional-arrays.ipynb** – Axes in multi-dimensional arrays
8. **8-3d-arrays.ipynb** – Working with 3D arrays
9. **9-vectorization-broadcasting.ipynb** – Vectorization and broadcasting in NumPy
10. **10-vector-normalization.ipynb** – Simple Normalizing vectors using NumPy
11. **11-numpy-math.ipynb** – Math functions in NumPy
